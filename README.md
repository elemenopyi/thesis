# Stop Perception gradiency modelling

## Overview
The study investigates how an LSTM models human gradient ratings and interpret cue importance
for Assamese-English bilingual listeners' stop contrast perception. 


## Repository structure
- `src/` – Scripts for experiment, data preprocessing, LSTM training, and analysis.
- `stimuli/` – Audio stimuli used in the perception experiment.
- `data/` – Raw and processed data.
- `requirements.txt` – Python dependencies.
- `README.md` – This file.

## Getting started
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (macOS/Linux) or `venv\Scripts\activate` (Windows).
4. Install dependencies: `pip install -r requirements.txt`
5. Explore scripts in the `src/` folder.
